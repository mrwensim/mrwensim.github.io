Xilinx-EES-260学习笔记：Microblaze软核系统搭建

  • Xilinx
  • microblaze
  • 软核
FPGA

记录了 Xilinx EES-260 建立 microblaze软核的过程。

打开 Xilinx Platform Studio(XPS) 软件。

点击 Create New Project Using Base System Builder。

选择好工程存放目录，然后点击 OK 。

根据FPGA开发板配置，然后点击 Next 。

点击 Add Device 添加外设。

选择 S6_DDRX_MEMORY 和 MCB_DDR2，点击 Apply。

再选择 UART 和 RS232，点击 Apply。

出现如图界面，点击 Finish 完成microblaze软核和基本外设的添加。

在 Bus Interface 窗口下找到 MCB_DDR2 并双击。

Memory Type 选择 LPDDR,点击 Next。

选择 MT46H32M16XXXX-5,之后不作修改，一直点击 Next。

在 Bus Interface 窗口找到 clock_generator_0 并双击。

将 CLKOUT_0 和 CLKOUT_1 的频率都改为 400000000(400MHz)，点击 OK。

点击上方 Ports 标签，右键点击 CLKIN，选择 No Connection。

右键点击 External Ports 下的 CLKN，选择 Delete External Port 将其删除。

同样地将 CLKP 删除。

至此，microblaze软核系统的搭建完成。
